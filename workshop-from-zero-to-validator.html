Workshop: From Zero to Validator
In this workshop we are going to learn how to create a new cosmos network and how to operate a validator.
The main points that we are going to see are:
How to create a wallet.
How to be part of the genesis validator set.
How to keep your validator keys secured (tmkms).
How to create a genesis file.
Assign coins to each validator.
Create validator genesis transactions.
Set up custom voting periods (this will be useful to create and vote on proposals without having to wait 5 days).
How to launch a cosmos chain.
How to check the consensus status (useful to know what validators are online).
How to claim rewards.
How to create an param-change proposal.
How to vote on a proposal.
How to create an upgrade proposal.
How to upgrade your node.
How to apply security upgrades.
